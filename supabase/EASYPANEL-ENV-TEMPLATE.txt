1. No Easy Panel, vá para seu projeto
2. Clique em "Environment Variables" ou "Variáveis de Ambiente"
3. Modo BULK EDIT (recomendado):
   - Clique em "Bulk Edit" ou "Edição em Massa"
   - Cole TODAS as variáveis acima (já com valores preenchidos)
   - Clique em "Save" ou "Salvar"

4. Ou modo MANUAL:
   - Adicione uma por uma usando o botão "Add Variable"
   - Nome: POSTGRES_PASSWORD
   - Valor: (cole o valor do seu .env)
   - Repita para todas as variáveis

Após configurar as variáveis:

1. Configure o Docker Compose File: easypanel.yml
2. Configure os domínios:
   - API: api.seu-dominio.com → porta 8000
   - Studio: studio.seu-dominio.com → porta 3000
3. Clique em Deploy
4. Aguarde todos os serviços ficarem green (Running)
5. Acesse o dashboard: https://studio.seu-dominio.com
6. Teste a API: https://api.seu-dominio.com/rest/v1/
